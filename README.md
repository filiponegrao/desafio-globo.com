# API Server

Simple Rest API using gin(framework) & gorm(orm)

## Endpoint list

### Bookmarks Resource

```
GET    /bookmarks
GET    /bookmarks/:id
POST   /bookmarks
PUT    /bookmarks/:id
DELETE /bookmarks/:id
```

### Users Resource

```
GET    /users
GET    /users/:id
POST   /users
PUT    /users/:id
DELETE /users/:id
```

server runs at http://localhost:8080
